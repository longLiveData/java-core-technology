谈谈你对java平台的理解？

Analysis：
问题笼统，题目开放。

Answers：
【java平台】
基本语言特性：面向对象(封装 继承 多态)，反射，泛型
java类库：核心类库，安全类库，jdk managment等类库，第三方类库
java虚拟机：垃圾收集器，运行时，动态编译，辅助功能
工具：辅助工具，编译器，诊断工具
java/JVM生态：JavaEE,Spring,Hadoop,Spark,Maven
【编译运行】
java源代码经过javac编译成.class文件
.class文件经JVM解析或编译运行：
解析：class文件经过JVM内嵌的解析器解析执行
编译：存在JIT(Just in Time Compile即时编译器)把经常运行的代码作为“热点代码”编译与本地平台相关的机?码，并进行各种层次的优化
AOT编译器：Java9提供的直接将所有代码编译成机?码执行
【和C/C++区别】
Write once，run anywhere
C/C++面向操作系统，需要开发者极大地关心不同操作系统之间的差异性
Java平台通过虚拟机屏蔽了操作系统的底层细节，使得开发者无需过多地关心不同操作系统之间的差异性
通过增加一个间接的中间层(JVM)来进行“解耦”是计算机领域非常常用的一种“艺术手法”
